# My Alert Blueprint

## Installation of this Blueprint

- Install button-card from HACS

- Copy the content of the Blueprint.yaml file into the Blueprint YAML code.

 ## Fields to defines.
 - The alert name
 - The alert entity
 - The alert icon when is OFF
 - The alert icon when is ON
